| Word | Example Sentence |

| **Unit 63: Cause, reason, purpose and result** | |
| **Cause and reason** | |
| because / since / as | I need to run **because / since / as** I'm late for the bus. |
| due to / caused by | The collision was **due to / caused by** the icy conditions. |
| the cause of | **The cause of** the collision was the ice on the road. |
| cause | The heavy rain **caused** the river to flow. |
| owing to | **Owing to** the icy conditions, the two cars collided. |
| spark (off) | The rise in prices **sparked (off)** a lot of political protest |
| give rise to / provoke / generate | The President's statement **gave rise to / provoked / generated** a lot of criticism |
| bring about / lead to | The new law has **brought about / lead to** great changes in education. |
| stem from | This problem **stems from** the inflation of recent years |
| arise from / arise out of | The court case **arose from / arose out of** allegatsion made in a newspaper. |
| **Reasons for and purposes of doing things** | |
| motive | I wonder what his **motives** were in sending that email? |
| prompt | I wonder what **prompted** her to send that message?
| with the aim of | She wrote to the press **with the aim of** exposing the scandal. |
| with a view to | I've invited you here **with a view to** resolving our differences. |
| on the grounds that | He refused to answer **on the grounds that** his lawyer wasn't there. |
| the purpose of | **The purpose of** her visit was to inspect the equipment. |
| **Results** | |
| as a consequence / consequently | He did not work. **As a consequence / Consequently**, he failed his exams. |
| consequence | the **consequence** of all these changes is that no one is happy any more. |
| result in | his remarks **resulted in** everyone getting angry. |
| outcome | The events had an **outcome** that no one could have predicted. |
| upshot | The **upshot** of all these problems was that we had to start again. |
| ensue | When the election results were announced, chaos **ensued**. |
| **Unit 64: Concession and contrast** | |
| **Verbs of concession** | |
| although | **Although** they were poor, they were independent. |
| nevertheless | He is a bit stupid. He's kind, **nevertheless**. |
| acknowledge / accept | I **acknowledge / accept** that he has worked hard but it's still not enough. |
| admit | I **admit** that I was wrong, but I still think we were right to doubt her. |
| concede | I **concede** that you are right about the goal, but not the method. |
| **Adverbs and other phrases showing contrast** | |
| That's all well and good / That's all very well | Ok, you're sorry. **That's all well and good / That's all very well**, but how are you going to pay us back? |
| after all | You shouldn't seem so surprised. **After all**, I did warn you. |
| for all that | He is boring, but, **for all that**, he is your uncle and we should invite him. |
| admittedly | **Admittedly**, she put a lot of effort in, but it was all wasted. |
| the reverse was true | I though she didn't like me. In fact, **the reverse was true**. |
| in contrast | In Europe they use the metric system. **In contrast**, the USA still uses many non-metric measurements. |
| on the other hand | It's not actually raining now. **On the other hand**, it may rain later, so take the umbrella. |
| **Collocating phrases for contrast** | |
| great divide | There's a **great divide** between city residents and people who live in the country. |
| huge discreptancy | There's a **huge discreptancy** between his ideals and his actions. |
| **Unit 65: Addition** | |
| **Expressions for linking sentences/clauses** | |
| in addition / additionally | **In addition / Additionally** to a degree, you need at least two years' experience for this job. |
| furthermore / moreover / what is more | Smartphones are becoming more sophisticated, **furthermore / moerover / what is more**, they are becoming cheaper. |
| equally / likewise | Children should respect their parents, **equally / likewise** they should respect their teachers. |
| on top of (all) that | We will have all the stress of going to court and giving evidence, **on top of that** we will have to pay the lawyer's bills. |
| plus | To keep fit you need a good diet, **plus** (you need) regular exercise. (inf.) |
| **Expressions at the end of sentences/clauses** | |
| etc. (et cetera) | They sell tables, chairs, beds, **et cetera**. |
| **Expressions that begin, or come in the middle of, sentences/clauses** ||
| further to | **Further to** my telephone call yesterday, I am now writing to ... (formal opening to an email) |
| in addition to | **In addition to** winning the gold medal, she also broke the world record. |
| alongside | **Alongside** her full-time job as a dentist, she does voluntary conservation work on weekends. |
| as well as | I have to go to the bank today **as well as** getting my hair cut. |
| besides / apart from | **Besides / apart from** having a salary, he also has a private income. |
| along with / together with | Jo Evans was there **along with / together with** a few other people who I didn't know. |
| **Unit 66: Referring Words** | |
| **Text-referring words** ||
| problem | Pollution is increasing. The **problem** is getting worse each day. |
| issue | Should taxes be lowered? This was the biggest **issue** in the election. |
| question | Whether the war could have been avoided is a **question** that interests historians. |
| topic | Let's discuss crime. That's always an interesting **topic**. |
| aspect | Punishment is only one **aspect** of crime. |
| **Problem-solution words** | |
| situation | The **situation** in our cities with regard to traffic is going from bad to worse. |
| problem | Congestions is a daily feature of urban life. This is a **problem**. |
| affect | The problem is now beginning to **affect** our national economies. |
| approach to a solution to a dilemma | Unless a new **approach** is found to control traffic, we will never find a *solution* to the *dilemma*. |
| claim | Your **claim** that we're doing nothing to invest in industry is false. |
| (this) fact | You've ignored this **fact**. |
| (that) argument | That **argument** is out-of-date in a modern technological society. |
| position | Our **position** has always been that we should encourage technology. |
| real point | The investment has all gone to service industries. The **real point** is that we need to invest in out manufacturing industries. |
| (that) view | That **view** won't help to reduce unemployment. |
| situation / state of affairs / position | This **situation / state of affairs / position** makes it hard for our company to generate profits. |
| problem / difficulty / crisis / matter | This **problem / difficulty / crisis / matter** needs to be addressed immediately. |
| response / reaction / attitude / approach (to) | I think this **response / reaction / attitude / approach (to)** our problem will harm our company in the long run. |
| solution / answer / resolution / key (to) | I think this is the optimal **solution / answer / resolution / key** to our problem. |
| evaluation / assessment / judgement | We will bring in a professional for the **evaluation / assessment / judgement** of this issue. |
| **Unit 68: Linking words in writing** | |
| **Organizing a formal text** | |
| first / firstly / first of all | **First / Firstly / First of all**, we must consider... |
| next | **Next**, it is important to remember that... |
| secondly / thirdly | **Secondly / Thirdly**, I would like to introduce... |
| finally / lastly | **Finally / Lastly**, we should recall that... |
| turning to | **Turning to** the question of foreign policy... |
| leaving aside | **Leaving aside** the question of pollution, there are also other reasons... |
| in parenthesis | **In parenthesis**, let us not forget that... |
| in summary / to sum up | **In summary / To sum up**, we may state that... |
| in sum | **In sum**, the economic issues are at the centre of this debate... |
| in conclusion / to conclude | **In conclusion / to conclude**, I should like to point out that... |
| **Linking words for explaining, exemplifying, rephrasing etc.** ||
| that is to say | To learn new words properly a lot of recycling is needed; **that is to say**, you have to study the same words over and over again. |
| for example / for instance | Some English words are hard to pronounce, **for example / for instance**, "eighth". |
| say | It might be possible, **say**, to include the parents in the discussion. |
| briefly | The Parliament has different committees. **Briefly**, these consist of two main types. |
| so to speak / as it were | She is, **so to speak / as it were**, living in a world of her own. |
| **Referring backwards and forwards in the text** | |
| the following | **The following** points will be covered in this essay: ... |
| above / earlier | It was stated **above / earlier** that the history of the USA is... |
| see | **See** page 238 for more information. |
| below | Many writers have claimed this (see **below**). |
| overleaf | A full list is given **overleaf**. |
| further | For **further** details / discussion, see Chapter 4. |
| refer you to | May I **refer you to** page 3 of my last letter to you? |
| with reference | **With reference** to your email of 12th March, ... |
